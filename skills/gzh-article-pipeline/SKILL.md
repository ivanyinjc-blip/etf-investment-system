# 公众号文章全流程生产流水线

> 从选题到草稿箱，全链路自动化。
> 核心优势：跨领域交叉选题思维 × HTML保留格式排版 × 图片自动嵌入 × 一键推送

---

## 流水线全景

```
选题矩阵 → MD撰写 → HTML排版(_wx.html) → 封面图(GPT-Image2) → 配图准备
                                                    ↓
                                              推送草稿箱(wx_push_article.py)
                                                    ↓
                                              公众号后台 → 预览 → 发布
```

---

## 一、目录结构规范

每篇文章一个独立文件夹，放在 `公众号/` 下：

```
公众号/
└── A04_斜杠命令/           ← 编号_简称
    ├── A04_斜杠命令.md      ← 原始Markdown（可选）
    ├── A04_斜杠命令_wx.html ← ★ 微信适配版（全inline style）
    ├── 封面图.png/jpg       ← 公众号封面（2.35:1，900×383推荐）
    ├── A04_配图01.png       ← 正文截图1
    ├── A04_配图02.png       ← 正文截图2（按需）
    └── A04_产品卡片.png     ← 产品/总结卡片（可选）
```

**命名规则**：
| 文件 | 格式 | 示例 |
|------|------|------|
| 微信HTML | `{编号}_{简称}_wx.html` | `A04_斜杠命令_wx.html` |
| 封面图 | `封面图.png` 或 `{编号}_封面.png` | `B02_封面.png` |
| 配图 | `{编号}_配图{序号}.png` | `A04_配图01.png` |
| 产品卡片 | `{编号}_产品卡片.png` | `B02_产品卡片.png` |

## 二、HTML排版规范（_wx.html）

### 2.1 必须遵守的规则

1. **全部inline style** — 不能用外部CSS、不能用`<style>`标签（微信会剥离）
2. **字体**：`PingFang SC, Hiragino Sans GB, Microsoft YaHei, sans-serif`
3. **背景色**：`#FDFBF7`（暖白纸色）
4. **正文字号**：`16px`，行高`1.85`
5. **代码块**：等宽字体 `JetBrains Mono, Fira Code, Consolas, monospace`
6. **最大宽度**：`680px`居中容器

### 2.2 必须包含的结构标记

```html
<!-- 封面图占位 -->         ← 推送时自动删除（封面只用thumb，不进正文）
<div>🖼️ 封面图待替换</div>

<header>                    ← 推送时自动删除（微信标题栏已显示）
  系列名 / H1标题 / 标签badge
</header>

<article>
  <h1>标题</h1>            ← 推送时自动删除（与微信标题栏重复）
  <blockquote>
    标签行 → 自动删除
    正文引言 → 保留
  </blockquote>
  正文内容...
</article>

<footer>                    ← 推送时自动删除（正文已有结尾语）
  点赞/在看/转发
</footer>
```

### 2.3 结尾规范

**作者署名**（最后一段，加粗）：
```html
<strong style="color:#2C2C2C;font-weight:700">我是阿超，一个认真在AI时代分享经验和思考的普通人。</strong>
```

**互动引导**（署名前一段，加下划线）：
```html
<u>如果觉得有用，欢迎点赞、在看、转发。</u>
```

### 2.4 截图占位符格式

正文中用以下格式标记截图位置：

```
[截图：01 - 命令总览速查表，管理类+开发类]
[截图：02 - .claude/commands/ 目录结构]
```

推送脚本会自动匹配 `配图01.png`、`配图02.png` 并替换为微信CDN图片。

## 三、封面图生成（GPT-Image2方法论）

### 3.1 核心原则

- **文字是绝对主视觉**，占据画面50%+面积
- **先理解词义→深层隐喻转译→再决定视觉风格**
- **每个标题生成不同的视觉人格**，拒绝模板化
- **画幅比例由词义决定**（知识类用横版2.35:1）

### 3.2 提示词结构

参考 `references/cover-prompt-template.txt`，核心模块：

| 模块 | 内容 |
|------|------|
| ①文字突出 | 10条硬标准：尺寸/位置/对比/不被遮挡 |
| ②隐喻转译 | 拒绝表层符号，寻找深层视觉思想 |
| ③字形表意 | 拉伸/断裂/嵌套，但保证可读 |
| ④隐喻系统 | 少量精准元素，围绕文字服务 |
| ⑤风格自适应 | 26种风格由词义驱动 |
| ⑥高级感 | 来自克制和准确，不来自堆元素 |
| ⑦构图 | 10种构图模式，文字永远第一层级 |
| ⑧画幅 | 1:1/3:4/16:9/21:9由词的"空间倾向"决定 |
| ⑨色彩光影 | 由词义决定配色，光影服务文字 |
| ⑩材质 | 纸张/金属/石材/墨迹/网格... |

### 3.3 操作流程

1. 分析标题深层隐喻 → 写分析（概念类型/气质/风格/色彩/构图）
2. 将分析+模板提示词粘贴到ChatGPT(GPT-4o)
3. 出图后下载到文章文件夹，命名为 `封面图.png`
4. 后续同session只需输入新标题即可换主题

## 四、推送脚本

### 4.1 wx_push_article.py（通用图文推送）

**位置**：`scripts/wx_push_article.py`

**功能**：
- 扫描文件夹中所有png/jpg → 上传微信永久素材
- 自动清理HTML重复元素（header/H1/标签行/footer/封面占位）
- 匹配`[截图：XX - 描述]` → 替换为实际图片
- 封面图只做thumb_media_id，不进正文
- 产品卡片自动插入指定锚点位置
- 创建图文草稿到公众号草稿箱

**用法**：
```bash
python3 scripts/wx_push_article.py "公众号/A04_斜杠命令" --author "阿超AI"

# 自定义参数
python3 scripts/wx_push_article.py "公众号/B01_τ定律" \
  --title "自定义标题" \
  --author "阿超AI" \
  --digest "自定义摘要"
```

### 4.2 wx_draft_image.py（单图片消息推送）

**位置**：`scripts/wx_draft_image.py`

**功能**：单张图片 → 上传 → 创建newspic草稿（小绿书）

**用法**：
```bash
python3 scripts/wx_draft_image.py 封面图.png "图片标题"
```

### 4.3 环境变量

```bash
export WX_APP_ID="wx945e86e48d0bab64"
export WX_APP_SECRET="你的AppSecret"
```

⚠️ AppSecret敏感，不要提交到Git。

## 五、完整操作流程

### 从头开始一篇新文章

```bash
# 1. 创建文章文件夹
mkdir -p "公众号/B04_新文章"

# 2. 写Markdown → 转微信HTML（手动或用小龙虾）
# 保存为 B04_新文章_wx.html
# 确保有：封面图占位 + header + article + 截图占位符

# 3. 生成封面图
# 复制 references/cover-prompt-template.txt 到ChatGPT
# 先贴隐喻分析，再贴提示词，目标词填文章标题
# 下载封面图到文件夹

# 4. 准备配图（截图/产品卡片）
# 放在同一文件夹，按配图01/02命名

# 5. 一键推送
python3 scripts/wx_push_article.py "公众号/B04_新文章" --author "阿超AI"

# 6. 去公众号后台草稿箱检查 → 预览 → 发布
```

### 已有文章重新推送

```bash
# 更新了封面图或配图后
python3 scripts/wx_push_article.py "公众号/B02_T型人才" --author "阿超AI"
```

## 七、常见问题

### Q: 推送后发现标题重复？
A: 检查_wx.html是否有`<header>`和`<article>`内的`<h1>`——脚本会自动清理，但确保结构标记正确。

### Q: 封面图没显示？
A: 检查文件夹里是否有`封面图.png`或`封面图.jpg`或`{编号}_封面.png`。

### Q: 配图没替换？
A: 占位符格式必须是`[截图：XX - 描述]`，且配图命名为`配图XX.png`。

### Q: AppSecret泄露了怎么办？
A: 去 https://mp.weixin.qq.com → 基本配置 → 重置AppSecret。
