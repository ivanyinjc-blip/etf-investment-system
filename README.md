## 核心架构

```
五态识别（ADX划分）
├── ADX ≤ 10    → 强反弹（RSI<40 → 8成仓）
├── ADX 10~15   → 弱反弹（RSI<40 → 5成仓）
├── ADX 15~25   → 震荡市（触布林下轨+RSI<40 → 7成仓）
├── ADX 25~35   → 弱趋势（轻仓跟进）
└── ADX > 35    → 强趋势（均线动量 → 满仓）

三层止损
├── 硬止损：收盘价 < 入场价×0.97（-3%，绝对红线）
├── ATR软止损：入场价 - 2.5×ATR(14)（趋势确认前）
└── 移动止损：MA5×0.98/0.96（趋势确认后ADX>35）
```

## 数据架构

- **tushare.db (DuckDB)** — 历史日线，267万条，覆盖全市场ETF
- **market_etf.db (SQLite)** — 20只重点ETF指标缓存
- **etf_trading.db (SQLite)** — 持仓、信号池、交易记录
