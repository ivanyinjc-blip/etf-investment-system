## 核心机制

### 第一层：硬止损（绝对红线，无条件触发）
```
触发条件：收盘价 < 入场价 × 0.97（-3%）
操作：次日开盘清仓
```

### 第二层：ATR软止损（趋势确认前使用）
```
触发条件：收盘价 < 入场价 - 2.5 × ATR(14)
操作：减仓50%
```

### 第三层：移动止损（趋势确认后，ADX>35）
```
触发条件1：MA5 × 0.98（减仓50%）
触发条件2：MA5 × 0.96（再减50%）
触发条件3：MA5死叉MA20（清仓）
```

## 最少持有天数规则（防震荡洗出）

```
强制持有：10个交易日（从买入次日起算）
例外条款（满足任一条即可提前减仓）：
1. 连续2天BIAS(5) < -3%
2. 大盘单日跌幅 > 3%
3. 浮盈达到 +15% 止盈目标且五态非强趋势（ADX≤35）
```

## 分批止盈体系

| 浮盈档位 | 触发条件 | 非强趋势(ADX≤35) | 强趋势(ADX>35) |
|:---|:---|:---|:---|
| 第一档 | +10% | 减仓50% | 不卖，继续持有 |
| 第二档 | +20% | 减仓70% | 不卖，继续持有 |
| 第三档 | +30% | 全部止盈 | 减仓50%，留仓奔跑 |

**趋势延续信号（出现任一则不止盈）：**
- ADX从低位上升超过10个点
- +DI > -DI 超过10个点（多头主导加强）
- RSI始终不破60
- 缩量不破MA5

## 每日检查清单（收盘后15:05前执行）

```
□ 硬止损触发？→ 当日清仓（最高优先级）
□ ATR软止损触发？→ 减仓50%
□ 最少持有期未到期？→ 禁止主动卖出
□ 趋势确认（ADX>35）？→ 是则切换移动止损体系
□ 移动止损触发？→ 按档位减仓
□ 止盈档达到但趋势延续？→ 不执行，继续持有
□ 无明确信号？→ 继续持有，不操作
```

## 加仓节奏纪律

### 核心原则：加仓必须等"关键位置"+"反转信号"双重确认

```
下跌加仓的充要条件（两者必须同时满足）：
1. 价格达到关键支撑位（布林下轨 / 前期低点 / 重要均线支撑）
2. 出现反转信号（RSI<40 / 底背离 / 锤子线 / 大阳线）
```

### 加仓触发条件（精确版）
```
条件A - 关键位置（满足任一即可）：
  □ 价格触及布林带下轨（BB_lower）
  □ 价格回踩20日均线支撑
  □ 创出近期新低后缩量企稳

条件B - 反转信号（满足任一即可）：
  □ RSI(14) < 40 超卖
  □ MACD底背离（价格新低但MACD未新低）
  □ 出现锤子线/吞没形态
  □ 连续3天下跌后出现缩量小阳线

加仓操作：满足A+B → 次日开盘加仓，仓位≤原持仓50%
```

### 禁止加仓情形（必须忍住）
```
✗ 跌了就加（无节奏）→ 结果：抄底抄在半山腰
✗ RSI还没到40就加 → 结果：左侧交易失败
✗ 大盘还在下跌趋势中加 → 结果：接飞刀
✗ 重要支撑破位后加 → 结果：逆势死扛
```

```
│  操作数量:                                       │

├─────────────────────────────────────────────────┤
│  触发条件检查（列出满足的条件）:                  │

│  □ BOLL位置: 上轨/中轨/下轨/距下轨___%           │
│  □ 趋势: 多头排列/空头排列/震荡                  │
│  □ MA状态: 价格 vs MA5/MA10/MA20                │
│  □ 成交量: 放量/缩量                            │
│  □ 硬止损距离: ___%                             │
├─────────────────────────────────────────────────┤
│  操作理由（为什么要现在做，而不是明天后天）:      │
│                                                 │
├─────────────────────────────────────────────────┤
│  预期收益 vs 风险（止损位___ → 空间___:___）:   │
└─────────────────────────────────────────────────┘
```

**通过标准：**
```
✅ VERIFIER-C审查通过 → 可执行
❌ VERIFIER-C打回 → 禁止操作
⚠️ VERIFIER-C警告 → 谨慎执行，减仓50%
```

**禁止操作情形：**
```
✗ 没有触发条件（感觉要跌/感觉会涨）→ 禁止
✗ RSI在中间区域（40-60）无明确信号 → 禁止
✗ 刚卖就想买回 / 刚买就想卖出 → 禁止
✗ 大盘方向不明时逆势操作 → 禁止
✗ 持仓不到10天（无特殊原因）→ 禁止卖出
```

## 失败案例库

### 案例18：无信号卖出（562500机器人ETF，2026-05-15）

```
【背景】
持仓成本: 1.135（持仓81,000股）
卖出价格: 1.135（刚好回本）
卖出时机: 盘中看到账户变绿，恐慌卖出

【当时数据】
RSI: 约60-65（中性区域，非超买）
BOLL: 距上轨1.147还有空间
趋势: MA5>MA10>MA20，多头排列未破
无任何止损/止盈条件触发

【卖出后行情】
当日下午跌至1.120（-1.3%）
随后大幅反弹至1.187（+4.6%）
踏空幅度: +4.6%

【问题诊断】
❌ 卖出没有触发条件——"感觉跌了"不是信号
❌ 心理驱动：看到账户变绿恐慌卖出
❌ 违反了"无明确信号不操作"原则
❌ 触发了"刚卖就想买回"的禁止情形

【正确做法】
✅ 等RSI>80 或 价格触及BOLL上轨 才考虑卖出
✅ 等跌破BOLL下轨(1.120)且RSI<40才考虑止损
✅ 趋势不破（MA20不破）→ 持有不动
✅ 触发前任何波动 → 忽略

【教训】
"不是因为跌了才卖，而是看到明确的卖出信号才卖"
"忍住不卖比卖掉更难"
"本次损失的不是钱，是本该赚到的+4.6%收益"
```

### 案例17：见光死（563230卫星ETF，2026-05-14）

```
【背景】
事件: 卫星相关利好传闻/政策出台
买入后: 当日冲高回落，随后连续下跌

【问题诊断】
事件驱动型标的 → 利好落地=卖点非买点
追事件驱动 = 买在消息最乐观的时刻
```

### 案例16：提前抄底（通用教训）

```
【问题】
RSI=50就认为超跌，提前加仓

【正确做法】
RSI<40 + 触BOLL下轨 = 双重确认才加仓
单RSI=50 ≠ 超卖
```

### 案例19：RSI>70追高买入（159995华夏能源ETF，2026-05-15）

```
【背景】
买入价格: 2.351（当日高点附近）
买入RSI: 79.5（超买区域）
买入理由: "市场调整后反弹，情绪上涨"
买入数量: 20,000股

【当时数据】
RSI: 79.5（>70，超买）
BOLL上轨: 2.352（买入价几乎触及上轨）
BIAS20: +16.4%（严重正偏离）
趋势: 多头排列

【问题诊断】
❌ RSI>70时追高买入——违反纪律（RSI>70禁止买入）
❌ 买入价2.351 = BOLL上轨2.352——买在天花板
❌ 买入理由"情绪上涨"不是技术指标——无信号操作
❌ BIAS20=+16.4%——严重偏离均线，调整不到位

【正确做法】
✅ 等RSI<40 + 价格触及BOLL下轨 才考虑买入
✅ 等BIAS20回到±5%以内
✅ 买入理由必须具体（"RSI=35触下轨"），不能模糊（"感觉会涨"）

【教训】
"RSI>70不是买入理由，是卖出理由"
"买在BOLL上轨=买在最高点附近"
"'情绪'不是信号，具体技术指标才是"
```

## 五步决策标准（每次操作前必答）

```
第一步：大盘是否配合？
  → 指数环境、赚钱效应、资金面

第二步：是否是买点？
  → 技术面位置（布林/MA/前高）+ RSI状态 + 催化事件

第三步：买入多少比例？
  → 根据五态+信号强度（10%试探/30%确认/50%重仓/80%满仓）

第四步：买入后持有策略？
  → 目标位 + 持有逻辑 + 分批止盈计划

第五步：止盈/止损条件？
  → 具体价格 + 具体条件，绝对禁止随意操作
```
