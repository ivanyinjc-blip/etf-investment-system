# 订阅 Plus / Pro / Team

Claude Code 提供了多档订阅方案，适合从个人开发者到大型企业的各类用户。本文帮你搞清楚各方案差异，选择最适合你的那一档。

---

## 价格与功能差异详解

### Free（免费版）

- **适合人群**：轻度体验用户、尝鲜者
- **用量限制**：每月有限次数的 Claude 4 Sonnet 对话额度
- **额度用完后**：需等待下月重置，或切换使用自己的 API Key
- **注意**：免费版不支持 Claude 4 Opus

### Pro（$20/月）

- **适合人群**：个人开发者、独立工程师
- **核心优势**：无限使用 Claude 4 Sonnet + Opus，无月度额度限制
- **额外权益**：优先客服支持、详细用量仪表板
- **国内订阅**：需要国际信用卡（Visa/MasterCard）或 PayPal

### Team（$25/人/月，最低5人起）

- **适合人群**：小型团队（5人以上）
- **核心优势**：团队共享配额、团队仪表板
- **Slack 集成**：可绑定 Slack 频道，Bot 直接推送通知
- **最低消费**：即使只有 5 人，每月也需 $125

### Enterprise（定制报价）

- **适合人群**：中大型企业
- **核心优势**：SSO/SAML、1M 超大上下文、专属客户经理
- **价格**：通常 $50+/用户/月，需联系 Anthropic 销售团队

## 国内用户订阅操作步骤

由于 Claude Code 订阅涉及国际支付，国内用户需要解决以下问题：

### 准备材料

| 材料 | 说明 |
|------|------|
| 国际信用卡 | Visa、MasterCard 或带外币功能的银联卡 |
| 或 PayPal 账户 | 绑定国内银行卡即可 |
| 或 Anthropic API Key | 自助服务，绕开订阅 |

> **Tip**：**推荐先使用 API Key 方式**，熟练掌握 Claude Code 的用法后，再决定是否订阅。API Key 按量计费，体验更灵活。

### 订阅步骤（Pro 为例）

1. 访问 https://claude.ai/settings/plans
2. 点击 **Upgrade to Pro**
3. 选择 **Pro Plan（$20/月）**
4. 输入信用卡信息（Stripe 安全支付）
5. 确认订阅，扣费后即时生效

### 国内支付常见问题

| 问题 | 解决方案 |
|------|---------|
| 信用卡被拒 | 尝试激活外币支付功能，或联系银行开通 |
| 没有国际信用卡 | 使用 PayPal 或改用 API Key 按量付费 |
| 银行卡被拦截 | 在银行 App/网页临时开通境外线上消费 |
| 订阅后没生效 | 等待 1-2 分钟，刷新页面；或登出重新登录 |

## API Key vs 订阅：到底选哪个？

这是一个非常常见的困惑，下面用一张表帮你做决策：

| 维度 | API Key（按量付费） | 订阅（包月制） |
|------|-------------------|--------------|
| **计费方式** | 按 token 用量计费 | 固定月费 |
| **适合用量** | 不确定/间歇性使用 | 频繁/持续使用 |
| **成本可控性** | 用多少付多少，明细清晰 | 超额后仍可继续（不额外计费） |
| **Claude Opus** | ✅ 可用（消耗更多 token） | ✅ 可用（无限） |
| **适合人群** | 轻度用户、预算敏感者 | 重度用户、团队 |
| **新手友好度** | 需了解 token 概念 | 简单直观 |

### 决策建议

```
用量少（月<100元）→ API Key 更划算
用量大（月>600元）→ Pro 订阅更划算
重度使用（每天几小时）→ Team 订阅更值
```

> **Tip**：新手建议先用 API Key 体验 1-2 周，感受自己的用量水平，再决定订阅方案。

## 额度管理

### 理解 Context（上下文窗口）

Claude Code 的对话有一个 **上下文窗口**上限，超出后旧内容会被"遗忘"或触发压缩：

- **Free/Pro**：200K tokens（约 15 万中文字）
- **Enterprise**：可定制，最高 1M tokens

> **注意**：上下文窗口不是你每次能输入的最大文字量，而是**整个对话历史**（包括你输入的内容 + Claude 的回复）的总量。

### Rate Limits（速率限制）

| 方案 | 请求频率限制 |
|------|------------|
| Free | 较低（排队机制） |
| Pro | 较高，基本无感知 |
| Team | 更高，支持团队并发 |

### 查看用量

```bash
# 查看当前配额和用量
claude auth status

# 在网页端查看详细用量
# 访问：https://claude.ai/settings/billing
```

## Tip：推荐先试用 API Key，熟练后再订阅

```
✅ 建议路径：
1. 用自己的 API Key 体验 1-2 周
2. 记录每天的平均用量（观察 token 消耗）
3. 估算月费用，对比订阅价格
4. 决定是否订阅 → 选择 Pro/Team
```

这样可以避免：

- ❌ 冲动订阅后发现不适合自己
- ❌ 不了解 token 概念，被账单吓到
- ❌ Team 起步最低 5 人，结果团队没人用
